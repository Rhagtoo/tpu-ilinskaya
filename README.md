# TPU Ильинская — Geo-Intelligence Report Generator

Автоматизированная сборка аналитических презентаций в стиле bnMAP.pro: карты на OSM-подложке + текстовая аналитика → готовый PPTX.

**Участок:** 50:11:0050603:423 (центр: 55.8000, 37.2949)  
**Площадь:** ~20.6 га (206 700 м²)  
**Кадастровая стоимость:** 1 166 366 760 ₽

## Инструменты

### 1. `generate_maps_final_v2.py` — генератор карт (v3)

```bash
venv/bin/python generate_maps_final_v2.py
```

**Зависимости:** `staticmap pillow`  
**Что делает:** рендерит 7 статичных PNG-карт на OSM-тайлах:
- Маркеры POI (цветные кружки + белое кольцо для участка)
- Граница участка (красный контур — реальная геометрия из Росреестра)
- Заголовок сцены, легенда, cadastre-инфо

**Сцены (7):**

| # | Имя | Заголовок | POI | Zoom |
|---|-----|----------|-----|------|
| 1 | location | Об участке и местоположении | 1 | 14 |
| 2 | transport | Транспортная доступность | 3 + МКАД | 12 |
| 3 | education | Образование и ДОУ | 6 | 11 |
| 4 | sport | Спорт и рекреация | 5 | 12 |
| 5 | trade | Торговля | 5 | 13 |
| 6 | medical | Медицина | 4 | 11 |
| 7 | competitors_bg | Конкурентное окружение | 9 | 11 |

**Исправления v3 (v2-скрипт):**
- Medical: zoom=11, центр (55.752) — участок и Лапино оба в кадре
- Education: центр (55.740) — лучший баланс
- Маркер «Участок»: r=11 + белое кольцо 4px
- Шрифты 13px подписи, 17px заголовок
- Легенда с увеличенными отступами

### 2. `build_full.py` — сборка PPTX

```bash
venv/bin/python build_full.py
```

**Зависимости:** `python-pptx pillow lxml`  
**Что делает:** перерисовывает слайды 4–9 в существующей презентации, вставляя свежие карты:
- Слайды 4–8: location, transport, education, sport, trade (перезапись)
- Слайд 9: medicine (новый)
- Слайды 1–3, 10–29: не трогает

**Дизайн-система:**
- Слайд: 10×5.63" widescreen
- Шапка: #0D2137 (тёмно-синий) + бейдж bnMAP.pro
- Заголовок: 18pt bold Calibri, #191919
- Текстовый блок: скруглённый, 3.45×3.62", #F8F8F8 / #EDF4FA
- Карта: 5.55" wide справа
- Вывод: тёмно-синяя плашка внизу

### 3. `cadastre.py` — запрос кадастровых данных

```bash
python cadastre.py 50:11:0050603:423
```

**Зависимость:** `rosreestr2coord`  
**На выходе:** GeoJSON с полигоном участка, центр, стоимость, категория.
